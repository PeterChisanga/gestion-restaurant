À propos de l'application

Bienvenue dans un système de gestion de restaurant, une application web développée avec Spring Boot, un cadre Java robuste. Cette application utilise l'architecture Modèle-Vue-Contrôleur (MVC), garantissant une séparation claire des préoccupations. Elle prend en charge les opérations CRUD (Create, Read, Update, Delete) complètes, vous permettant de gérer les employés, les plats du menu et les commandes de manière transparente. Bootstrap est utilisé pour le style des vues, offrant une interface utilisateur moderne et réactive. N'hésitez pas à explorer et à utiliser les diverses fonctionnalités de l'application pour rationaliser les opérations de votre restaurant.
